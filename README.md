# dcd_back_end

github 地址： `https://github.com/zhiyouth/dcd_back_end`

## run project

- 本地安装 mysql
- 本地配置 mysql 环境变量 略
- 登录 mysql `mysql -uroot -proot;`
- 创建数据库 `creacte database dcd_mini_db;`
- 查看数据库 `show databases;`
- 使用数据库 `use dcd_mini_db;`
- npm i
- npm run start
